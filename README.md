## 💡 서비스 소개

BOOKIZ는 아이와 교감하면서 실감나게 동화를 읽어주는 서비스입니다. <br />
아이들에게 동화책을 직접 읽어줌으로써 얻을 수 있는 장점을 살리고, 동시에 부모님의 부담을 덜 수 있는 서비스를 기획하게 되었습니다.
